# 🚀 Quick Start - Dimel's School MVP

## Запуск за 30 секунд

```bash
# Установка зависимостей
npm install

# Запуск dev сервера
npm run dev

# Открыть http://localhost:5173
```

---

## 🎮 Как тестировать

### Вариант 1: Студент с нуля
1. Открыть `/`
2. Нажать "Регистрация"
3. Ввести:
   - Имя: Иван
   - Email: ivan@test.com
   - Пароль: 123
4. Попадете в dashboard без доступа
5. Нажать "Купить курс"
6. Выбрать любой тариф
7. Нажать "Оплатить"
8. Ждать 2 сек → Success
9. Курс открыт!

### Вариант 2: Админ
1. Открыть `/`
2. Войти с:
   - Email: `admin@test.com`
   - Пароль: любой
3. В header появится "Админка"
4. Перейти в админку
5. Доступны все разделы

### Вариант 3: Dev Helper
1. Зарегистрироваться как любой пользователь
2. В правом нижнем углу кнопка ⚙️
3. Нажать → откроется панель
4. Кнопка "Grant Course Access" → мгновенный доступ
5. Не нужно проходить оплату!

## 🆘 Частые вопросы

**Q: Почему не могу войти в админку?**
A: Email должен быть точно `admin@test.com`

**Q: Оплата не проходит?**
A: Это mock, просто ждите 2 сек. 10% шанс на failed.

**Q: Прогресс не сохраняется?**
A: Проверьте localStorage не заблокирован.

**Q: Где изменить данные курса?**
A: `/src/app/data/courseData.ts`

**Q: Как добавить нового админа?**
A: В `AuthContext.tsx`, строка 36: добавьте email в проверку

**Q: DevHelper не появляется?**
A: Только в dev режиме. В prod отключен.
